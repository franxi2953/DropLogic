# Example Scripts

El repositorio ya incluye scripts de ejemplo ejecutables dentro de la carpeta [`examples/`](https://github.com/franxi2953/DropLogic/tree/main/examples).

Esta subsección documenta los dos scripts principales referenciados en **Getting Started**:

- [`simulator_example.py`](https://github.com/franxi2953/DropLogic/blob/main/examples/simulator_example.py)
- [`DMLite_example.py`](https://github.com/franxi2953/DropLogic/blob/main/examples/DMLite_example.py)

Cada página de abajo incluye:

- una explicación breve de lo que demuestra el script
- el código fuente completo embebido
- un enlace directo al archivo dentro del repositorio público

## Ejemplos Incluidos

### Ejemplo de Simulator

Úsalo cuando quieras probar routing, planificación y visualización sin hardware.

### Ejemplo de DMLite

Úsalo cuando quieras ejecutar ese mismo tipo de flujo sobre una configuración real con matriz de electrodos `DMLite`.

## Notas

- El repositorio también contiene `advanced_usage.py`, pero actualmente es solo un placeholder.
- Estos ejemplos están pensados como puntos de partida legibles, no como flujos de producción cerrados.
