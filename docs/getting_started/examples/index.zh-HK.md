# 範例腳本

- [`simulator_example.py`](https://github.com/franxi2953/DropLogic/blob/main/examples/simulator_example.py)：純軟件 `Simulator` 系統範例

每個頁面都保留完整源碼，方便跟住做、複製同修改。

`advanced_usage.py` 目前仍然係 placeholder。
