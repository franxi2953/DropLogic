# Utilidades

Las utilidades son capas de soporte usadas por sistemas, planificacion, integracion hardware y depuracion.

No son el flujo principal de usuario, pero hacen que el flujo sea fiable:

- conversiones de coordenadas y helpers de configuracion
- gestion de calibracion
- deteccion de gotas y condensados
- helpers de captura de imagen
- depuracion de planes
- diagnosticos de runtime
- configuracion de logging

La mayoria de usuarios empieza por `Systems` y `Planning`, y viene aqui cuando necesita ajustar hardware, inspeccionar fallos o integrar modulos nuevos.
