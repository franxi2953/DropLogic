!!! info "平台兼容性"
    裝好相配嘅 DropLogic 原生 runtime 之後，原生 `DMLite` 硬件控制支援 Windows x86_64、macOS Apple Silicon、Linux x86_64、Raspberry Pi OS 64-bit 同 Raspberry Pi OS 32-bit。`Simulator` 仍然係純 Python，唔需要原生硬件 assets。

歡迎閱讀 **DropLogic** 文檔。DropLogic 係一個用於數碼微流控（DMF）控制嘅 Python 函式庫。佢用統一嘅 Python 介面封裝系統、模組、規劃、執行同視覺化，令腳本保持清楚，而唔需要喺唔同硬件介面之間切換。
