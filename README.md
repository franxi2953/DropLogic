# DropLogic Library - Digital Microfluidics Control Platform

A comprehensive Python library for controlling digital microfluidics (DMF) hardware systems with advanced droplet manipulation capabilities.

## Documentation

**For full installation instructions, system architecture, and API reference, please visit our official documentation website:**

**[https://franxi2953.github.io/DropLogic/](https://franxi2953.github.io/DropLogic/)**

## Quick Start

```bash
git clone https://github.com/franxi2953/DropLogic.git
cd DropLogic
pip install .
```

*Note: For native control of physical hardware devices, an additional DropLogic Runtime Installer may be required for your target platform. This installer is distributed separately from the Python library. Please refer to the [Getting Started Guide](https://franxi2953.github.io/DropLogic/getting_started.en) for details.*
